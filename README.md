Airbnb Data Analytics Project
📌 Overview
This project demonstrates end‑to‑end data analytics skills by working with an Airbnb dataset. It covers loading and exploring data in Python, cleaning and transforming it, running SQL queries across multiple database systems, and visualizing insights in a Power BI dashboard. The final deliverable is a professional report highlighting key findings.

📂 Dataset
Source: Airbnb listings dataset (CSV/Excel format)

Content: Property details, host information, pricing, location, and reviews

Size: Medium‑scale dataset suitable for EDA and dashboard creation

🛠 Tools & Technologies
Python (Pandas, NumPy, Matplotlib, Seaborn)

SQL (PostgreSQL, MySQL, SQL Server)

Power BI (interactive dashboards)

Jupyter Notebook / VS Code (development environment)

🔎 Steps

Inspect structure and data types

Exploratory Data Analysis (EDA)

Summary statistics

Distribution plots and correlations

Handle missing values

Create tables in PostgreSQL/MySQL/SQL Server

Compare performance across systems

Power BI Dashboard

Add slicers for interactivity

Report Creation

Summarize findings
